🚀 懒人推荐：双击一键运行（推荐）
--------------------------------------------------
为了获得最佳体验，我们已经为您配置好了自动化脚本：

1. 确保您的电脑上已经安装过 [Node.js]（未安装请前往 https://nodejs.org 下载常青 LTS 版本一键安装）。
2. 在本项目根文件夹下：
   * Windows 电脑用户：直接双击运行 【 start.bat 】 文件
   * Mac / Linux 电脑用户：在终端中运行 【 chmod +x start.sh && ./start.sh 】
3. 运行后，脚本会做以下两件酷炫的事：
   * 自动在您的 Windows 电脑桌面上创建一个名为【亦仁翻译一键启动】的桌面快捷方式，以后双击桌面图标就能直接启动了！
   * 自动帮您下载所有运行所需的依赖，并直接帮您在浏览器里打开翻译系统页面 ( http://localhost:3000 )。
